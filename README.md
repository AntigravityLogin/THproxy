# FanProxy - Расширение для Chrome

Расширение для Google Chrome, позволяющее в один клик подключать HTTP/SOCKS прокси с поддержкой нескольких форматов ввода строки.

## Установка
1. Скачайте архив расширения со страницы релиза: [FanProxy Release](https://github.com/AntigravityLogin/THproxy/releases/tag/Thproxy).
2. Распакуйте скачанный ZIP-архив в любую удобную папку.
3. Откройте Google Chrome и перейдите по адресу `chrome://extensions/`
4. Включите **"Режим разработчика"** (Developer mode) в правом верхнем углу.
5. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked).
6. Выберите только что распакованную папку.

## Использование
1. Нажмите на иконку расширения в панели браузера.
2. Вставьте вашу строку с прокси в одном из поддерживаемых форматов.
3. Нажмите кнопку **"Подключить прокси"**.
4. Весь трафик вашего профиля Chrome пойдёт через указанный прокси!

## Поддерживаемые форматы прокси
| Формат | Пример |
|---|---|
| `http://user:pass@ip:port` | `http://login:password@1.2.3.4:6427` |
| `http://ip:port:user:pass` | `http://1.2.3.4:6427:login:password` |
| `ip:port:user:pass` | `1.2.3.4:6427:login:password` |
| `ip:port` | `174.140.200.147:6427` |
| `socks5://user:pass@ip:port` | `socks5://user:pass@1.2.3.4:1080` |

## Особенности
* Поддержка нескольких форматов прокси-строк (URL, нестандартный, без пароля).
* Автоматическая обработка авторизации (логин:пароль).
* Поддержка изменения прокси «на лету» без перезапуска браузера.
* Сохранение последней введённой строки прокси для удобства.
* Встроенная IP-панель с геоданными и пингом сайтов.
